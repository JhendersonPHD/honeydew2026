# Honeydew Themes Calculator

This is a standalone React application for themes calculator, featuring an implemented routing system.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

## Routing

The routing is powered by `react-router-dom` v6 and supports:
- `/` - Home
- `/calculator` - Calculator
- `/themes` - Themes

## Testing

Run tests with:
```bash
npm test
```
