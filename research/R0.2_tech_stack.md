### Multi-Tenancy Patterns

**Option A: Schema-per-Farm (Prisma)**
- Farm-scoped tables with subdomain isolation
- Each farm has own products, orders, inventory

**Option B: Tenant ID Pattern**
- Middleware extracts `x-farm-id` header
- All queries scoped by farmId

### Inventory Management

```javascript
// Distributed inventory service pattern
class InventoryService {
  async reserveStock(farmId, productId, quantity) {
    return await prisma.$transaction(async (tx) => {
      // Lock row for update (prevents overselling)
      const inventory = await tx.$queryRaw`
        SELECT * FROM "Inventory" 
        WHERE "farmId" = ${farmId} AND "productId" = ${productId}
        FOR UPDATE
      `;
      
      if (inventory.quantity < quantity) {
        throw new InsufficientStockError();
      }
      
      await tx.inventory.update({
        where: { id: inventory.id },
        data: { quantity: { decrement: quantity } }
      });
      
      return { reserved: true, remaining: inventory.quantity - quantity };
    });
  }
}
```

## 3. Shopify API Integration Best Practices

### Integration Architecture

```javascript
class ShopifyInventoryService {
  constructor(credentials) {
    this.shopify = new Shopify({
      shopName: credentials.shopName,
      apiKey: credentials.apiKey,
      password: credentials.password,
      apiVersion: '2024-01'
    });
  }
  
  // Webhook handler for inventory updates
  async handleInventoryUpdate(webhookPayload) {
    const { variant_id, available } = webhookPayload;
    
    // Sync to local farm inventory database
    await prisma.inventory.updateMany({
      where: { shopifyVariantId: variant_id },
      data: { quantity: available }
    });
    
    // Trigger low stock alerts
    if (available < 10) {
      await this.checkLowStockAlert(webhookPayload);
    }
  }
  
  // Bulk sync products from Shopify to local catalog
  async syncProducts(options = {}) {
    const products = await this.shopify.product.list({
      limit: 250,
      ...options
    });
    
    return await prisma.$transaction(async (tx) => {
      for (const product of products) {
        await tx.product.upsert({
          where: { shopifyProductId: product.id.toString() },
          create: {
            shopifyProductId: product.id.toString(),
            name: product.title,
            description: product.body_html,
            price: product.variants[0].price,
            shopifyVariantId: product.variants[0].id.toString(),
            farmId: options.farmId
          },
          update: { /* ... */ }
        });
      }
    });
  }
}
```

### Key Shopify Integration Points
1. **Product sync** — Bi-directional inventory updates
2. **Webhook listeners** — Real-time inventory change notifications
3. **Conflict resolution** — Priority rules for stock levels
4. **Sync history** — Audit trail for debugging

## 4. AI-Powered Recommendation Systems

### Techniques for E-Commerce Recommendations

1. **Collaborative Filtering**
   - User-to-user similarity based on purchase history
   - "Customers who bought this also bought..."

2. **Content-Based Filtering**
   - Product attributes matching user preferences
   - "You usually buy leafy greens, here's what's fresh"

3. **Hybrid Approaches**
   - Combine collaborative + content-based
   - Context-aware (season, weather, location)

## 5. UX Best Practices for Farm/Fresh Food E-Commerce

### Cart Flow Optimization

1. **Persistent Cart** — Save cart across sessions (localStorage + DB sync)
2. **Quick Add** — One-click add from product cards
3. **Quantity Selector** — Prominent +/- buttons with direct input
4. **Running Total** — Always visible subtotal with shipping estimate
5. **Farm grouping** — Visual grouping by farm in multi-farm baskets

### Checkout Optimization

| Stage | Best Practice |
|-------|--------------|
| Cart | Farm grouping, quantity editing, promo codes |
| Address | Saved addresses, address autocomplete |
| Payment | Multiple options (card, PayPal, Apple/Google Pay) |
| Confirmation | Order details, farm notifications, tracking |

### Mobile-First Considerations
- Thumb-zone accessible navigation
- Quick-add from product listings
- Camera for receipt scanning
- Push notifications for delivery windows

### Freshness-Focused UX
- "Harvest date" prominently displayed
- "Best by" date on each item
- Seasonal badges/features
- Farm origin visualization on map
